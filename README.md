# FullStack Application

A complete fullstack application with React frontend, Node.js backend, MongoDB database, and JWT authentication.

## Features

- ✅ User authentication (Register/Login/Logout)
- ✅ JWT-based security
- ✅ Task management (CRUD operations)
- ✅ Search and filter functionality
- ✅ Responsive design with TailwindCSS
- ✅ Protected routes
- ✅ Password hashing

## Quick Start

### Backend Setup
```bash
cd backend
npm install
npm run dev
